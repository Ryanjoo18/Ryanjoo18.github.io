Fires and Hotspots

Download

The analysis and source code can be found at this GitHub repository.

Description

I analyzed fire hotspots in South Sumatra using Sentinel-2 and FIRMS datasets to identify high-confidence fire zones. I processed multi-spectral satellite imagery and used True Color, False Color (NIR), and SWIR composites to visualize smoke plumes and burned areas. I presented my findings at the Space Faculty Remote Sensing Workshop in 2023.
